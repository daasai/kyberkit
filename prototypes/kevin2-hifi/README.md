# Kevin 2.0 高保真静态原型

独立子项目：不依赖主仓库 `app/` 或 Sidecar。视觉参考 [Junior 品牌资源](https://junior.so/press)；信息架构与屏幕内容对齐 Kevin 2.0 文档（`docs/specs/kevin2.0/`）。

## 运行

```bash
cd prototypes/kevin2-hifi
npm install
npm run dev
```

开发服务器固定端口 **5100**：<http://localhost:5100>。左侧为场景切换，无后端。
